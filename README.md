## Prerequisites

* Linux host such as Arch Linux, Debian, Ubuntu, RHEL ...etc
* Python >= 3.10
* uv
* key-value cache
* PostgreSQL >= 17
* tree
    * a CLI tools to list contents of directories in a tree-like format.
* docker/podman (just pickup one to do conatiner image building and running)

## Quick Start

```shell
uv run main.py
```
